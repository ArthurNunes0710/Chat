# Implementação Chat via web

- Implementação de servidores e clientes em python

## Funcionalidades

- Comunicação com múltiplos usuários simultaneamente
- Registro de contas
- Sala global e salas privadas
- Mensagens guardadas em banco de dados

## Ferramentas utilizadas

- Flask
- SQLAlquemy
- PostgreSQL
- html
- css

## Deploy

- Realizado na plataforma Railway
- Configurado para 2 replicas
- Dockerfile informando códigos e bibliotecas necessárias
